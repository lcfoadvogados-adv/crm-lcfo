Política de Privacidade

LCFO Advogados — Sistema CRM Interno

Última atualização: maio de 2026

1. Coleta de Dados

Este sistema coleta dados de potenciais clientes (nome, CPF, telefone, e-mail e endereço) exclusivamente para fins de gestão interna do escritório LCFO Advogados. Os dados são obtidos por meio de formulários de anúncios do Meta Ads e cadastros manuais realizados pelos advogados do escritório.

2. Uso dos Dados

Os dados coletados são utilizados somente para:

  • Contato com o potencial cliente para oferta de serviços jurídicos;
  • Gestão interna de clientes e processos do escritório;
  • Cumprimento de obrigações legais.

3. Compartilhamento

Os dados não são compartilhados com terceiros, exceto quando exigido por lei ou ordem judicial.

4. Armazenamento

Os dados são armazenados em servidores seguros (Vercel / Upstash) com acesso restrito aos profissionais do escritório. Os dados ficam armazenados pelo período necessário para a prestação dos serviços.

5. Direitos do Titular

Nos termos da Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018), o titular dos dados pode a qualquer momento solicitar acesso, correção, exclusão ou portabilidade dos seus dados pelo e-mail: lcfoadvogados@gmail.com.
